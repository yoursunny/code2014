播客编辑器 - %title%

播客编辑器能让你更方便快捷的在页面上添加声音、影片对象！

播放器种类
Flash(推荐) 可插入SWF文件
Windows Media Player(推荐) 可插入MP3 WMA WMV ASF AVI MPG M3U文件
RealPlayer 可插入MP3 RM RMVB RA RAM MPG SMI SMIL文件
<bgsound>(只支持IE浏览器) 可插入MID MP3 WMA WAV文件
QuickTime 可插入MOV文件
(粉红色文件格式适合网络传送，尽可能采用；棕色文件格式不利于在线点播，请尽量不要使用。访问者系统中需安装相应插件才能播放，否则将不能播放)
自动开始播放(只用于Windows Media Player与RealPlayer与QuickTime)

播放器尺寸
宽 * 高 (请根据影音文件实际尺寸设置，可适当扩大)
参考：
Windows Media Player高度需设置为图像高度+45以容纳控制栏
Flash与RealPlayer设置为实际尺寸

播放文件
输入节目地址
或 上传节目 (大型文件无法成功上传)
(上传节目将覆盖节目地址；修改节目地址将删除已上传节目。节目地址不会保密，为了防止被多线程下载，建议直接FTP上传到专用点播服务器，而不要在此上传)
