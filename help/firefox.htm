安装Firefox浏览器 - PageGenerator

Mozilla Firefox是一个自由的、开放源码的浏览器，适用于 Windows、Linux 和 MacOS X平台。它体积小速度快，还有其它一些高级特征，主要特性有：标签式浏览，使上网冲浪更快；可以禁止弹出式窗口；自定制工具栏；扩展管理；更好的搜索特性；快速而方便的侧栏。

Firefox完全遵循W3C制定的国际互联网标准(HTML CSS JavaScript DOM等)，使用Firefox浏览器登录PageGenerator程序可以避免出现一些奇怪的兼容性问题。

注意：PageGenerator只要求管理者使用Firefox；对于所产生网站访问者的浏览器并没有严格的要求。
