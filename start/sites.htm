您应该知道的几个基本概念
站点
一个位于本地或FTP服务器、用本程序管理的网站或虚拟目录
目录
站点中的一个文件夹，每个目录有且只有一个页面，并且可以在目录中创建子目录
模板
页面外观的统一定义，规定了页面的框架、布局，以及能够插入区块的位置
区块
页面上的一个内容或功能模块，例如文本、图片、调查等，可以用相应编辑器创建和修改
列表
列出若干页面(目录)链接的页，通常用于显示于网站首页、栏目页；每个列表均提供HTML JavaScript XML(Atom聚合源)三种输出形式
