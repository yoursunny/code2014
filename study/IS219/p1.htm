Windows NT安全基础，Windows 2000安全基础 习题

下列哪些安全特性是Windows 2000引入的？

  • 安全审核
  • 活动目录
  • IPSec
  • EFS

用户模式中的代码可以直接访问所有的内存空间。

安全访问令牌由哪个组件生成？

  • winlogon
  • LSA
  • SRM
  • Object Manager

C2级别的安全性策略(4条)

  • 自由的访问控制
  • 对象的重用
  • 强制的用户标识和认证
  • 可记账性和审核

Windows2000域控制器是相互平等的，没有“主域服务器”和“备份服务器”的区分。

WindowsNT域newyork.contoso.com信任paris.contoso.com，paris.contoso.com信任beijing.contoso.com，则newyork.contoso.com信任beijing.contoso.com。

Windows2000安全性目标

  • 企业中的单一登录
  • 集成的安全服务
  • 管理的委派和可扩展性
  • 强大的身份验证
  • 用于实现互操作能力的基于标准的协议
  • 审核服务

SRM在什么时候进行安全性检查？

  • 系统启动时
  • 用户登录时
  • 创建对象句柄时
  • 访问对象时

Windows2000支持哪些安全协议？

  • NTLM
  • UAC
  • Kerberos
  • TLS
