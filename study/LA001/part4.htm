1、我国为什么选择人民代表大会制度作为我国的政体？

P129~130

人民代表大会制度的优越性

  • 第一，人民代表大会制度适合中国的国情，因而有很强的生命力。

2、什么是债？它有哪些特征？

P219~220

债是按照合同的约定或者依照法律的规定在当事人之间产生的特定的权利和义务关系。享有权利的人是债权人，负有义务的人是债务人。

债作为一种民事法律关系，它具有如下特征：

  1. 债的关系的当事人都是特定的。
  2. 债的关系的客体可以是物或知识产权，也可以是行为。
  3. 债权的实现必须依靠义务人履行义务的行为，义务人不履行行为，债权人的权利就不能实现。
  4. 从发生根据上看，债可以因合法行为而发生，例如因合同行为而转让财产；债也可以因不法行为而发生，比如因侵权行为所生之债。

3、什么是缓刑？其适用条件是什么？

P366~367

缓刑，是指人民法院对于被判处拘役、3年以下有期徒刑的犯罪分子，根据犯罪分子的犯罪情节和悔改表现，认为暂缓执行原判刑罚，确实不致再危害社会的，规定一定的考验期，在考验期限内不再犯新罪或者符合法定的其他条件，原判刑罚就不再执行的刑罚制度。

4、依法治国的基本含义是什么？

课件11页

依法治国就是广大人民群众在党的领导下，依照宪法和法律规定，通过各种途径和形式管理国家事务，管理经济文化事业，管理社会事务，保证国家各项工作都依法进行，逐步实现社会主义民主的制度化、法律化，使这种制度和法律不因领导人的改变而改变，不因领导人看法和注意力的改变而改变。

5、民事诉讼时效的中断规则是什么？

P248

诉讼时效中断是指在诉讼时效进行中，由于发生法定事由，使以前经过的时效期限统归无效，时效期间从中断之时起重新计算。法定事由包括下列3种：

  • 提起诉讼
  • 当事人一方提出履行要求
  • 义务人同意履行义务

P246

8、结婚的法定条件有哪些？

课件130页——EE版答案不完整，敬请注意

（一）结婚的实质条件

  1. 双方达到法定婚龄。
  2. 双方自愿。
  3. 符合一夫一妻制。
  4. 禁止直系血亲和三代以内的旁系血亲结婚。
  5. 禁止患有医学上认为不应当结婚的疾病者结婚。

（二）结婚的程序条件：要求结婚的男女双方必须亲自到婚姻登记机关进行结婚登记。符合婚姻法规定的，予以登记，发给结婚证。取得结婚证，确立夫妻关系。

9、继承人在什么情况下丧失继承权？

课件146页

继承人有下列行为之一的，丧失继承权：

  1. 故意杀害被继承人的；
  2. 为争夺遗产而杀害其他继承人的；
  3. 遗弃被继承的，或者虐待被继承人情节严重的；
  4. 伪造、篡改或者销毁遗嘱，情节严重的。

  2. 有明确的被告

12、违法的构成要件是什么？

P60

  1. 违法必须是人们违反法律规定的一种行为，包括作为和不作为。仅有思想而无行为不构成违法。
  2. 违法必须是在不同程度上侵犯法律所保护的社会关系的行为，必须是对社会造成一定危害的行为。
  3. 违法必须是行为者出于故意或过失，也就是行为人在主观上要有过错。
  4. 违法的主题必须具有法定责任能力。

13、《合同法》规定的无效合同有哪些？

P239

  1. 一方以欺诈、胁迫的手段订立合同，损害国家利益；
  2. 恶意串通，损害国家、集体或者第三人利益；
  3. 以合法形式掩盖非法目的；
  4. 损害社会公共利益；
  5. 违反法律、行政法规的强制性规定。

14、哪些智力成果依法不授予专利权？

P231

  1. 科学发现；
  2. 智力活动的规则和方法；
  3. 疾病的诊断和治疗方法；
  4. 动物和植物品种，但动物和植物品种的生产方法可以授予专利权；
  5. 用原子核变换方法获得的物质。

15、婚姻法规定的效力瑕疵的婚姻有哪些？

1、无效婚姻的情形

  • 重婚的；
  • 有禁止结婚的亲属关系的；
  • 婚前患有医学上认为不应当结婚的疾病，婚后尚未治愈的；
  • 未达到法定婚龄的(宣告无效的申请应在法定婚龄届至前提出)。

2、可撤销婚姻

  • 因胁迫结婚的(受胁迫方1年内有可请求撤销)。
无效和被撤销的婚姻，自始无效。

17、哪些财产属于夫妻一方的个人财产？

P262——此题题目与EE版答案中的题目不同，可能是他们粘贴错了

有下列情形之一的，为夫妻一方的财产，双方另有约定的除外：

  • 一方婚前所有的财产
  • 因一方身体受到伤害获得的医疗费、残疾人生活补助费等费用
  • 遗嘱或赠与合同中指明归一方的财产
  • 一方专用的生活用品
  • 其他应当归一方的财产

19、发明创造的专利申请权如何归属？

课件103页

  1. 职务发明创造的专利申请权归单位。——但是，职务作品的著作权归作者，要注意区分
  2. 非职务发明创造的专利申请权归发明人或设计人。但利用本单位物质技术条件完成的发明创造，单位与发明人或设计人对专利申请权和专利权的归属另有约定的，从其约定。
  3. 两个以上单位或个人合作完成的发明创造，除另有协议外，专利申请权归共同完成的单位或个人。
  4. 委托完成的发明创造，除当事人另有约定外，专利申请权归研究开发人。
  5. 专利申请权依法转让、继承后，归受让人、继承人。
