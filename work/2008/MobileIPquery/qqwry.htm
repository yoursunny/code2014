手机号码IP地址归属地查询

纯真IP数据库

纯真IP数据库是金狐软件维护的IP归属地数据。

纯真IP数据库的文件名是QQWry.dat，每5天更新一次；QQWry.dat下载地址。该数据库侧重中国、甚至精确到网吧机位，其他国家很简略。

开发应用系统时，可以编程查询纯真IP数据库。纯真IP数据库并不是Access、SQLite等常见数据库格式，而是一个特殊的二进制格式；纯真IP数据库格式详解。

格式很难懂，不是吗？如果你使用的常见的编程语言，完全可以采用现成的查询代码：

C#
QQWry.NET by 阿不
IPScanner by rexsp（效率不如QQWry.NET）
PHP
IpLocation by 马秉尧
Python
IPLocator by echo.xjtu
Java
IPSeeker by lumaqq
C/C++
IPLocator by echo.xjtu

一个日趋严重的问题是：随着收录的增加，纯真IP数据库变得越来越大了。有时候，并不需要如此详细的归属地数据。我们还有替代的选择：珊瑚虫IP数据库，文件名为CoralWry.dat，它的格式与纯真一致，数据作了精简，体积小很多；珊瑚虫IP数据库下载地址

纯真网络还提供了几个客户端脚本，可以嵌入网页显示IP地址。可是近期纯真网站不能正常访问，因此无法使用。

显示当前访问者IP的归属地
<script type="text/javascript" src="http://www.cz88.net/ip/ipcheck.aspx"></script>
效果：无法演示
注意：该脚本使用了document.write，与XHTML冲突；XML+XSLT网页无法使用，否则……用Firefox看看就知道了
